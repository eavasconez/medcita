# MedCita — MVP Beta

MedCita es una solución simple de agenda médica.

## 🚀 Instrucciones en 3 pasos

### 1. Configuración de Backend
```bash
cd backend
npm install
# Asegúrate de tener un archivo .env si deseas usar Twilio real
npm run dev
```

### 2. Configuración de Frontend
```bash
cd frontend
npm install
npm run dev
```

### 3. Acceso a la Demo
- **URL**: http://localhost:5173
- **Email**: demo@medcita.ec
- **Password**: demo1234

## 🛠️ Stack
- **Frontend**: React (Vite) + Tailwind CSS
- **Backend**: Node.js + Express
- **DB**: SQLite (archivo local `database.sqlite`)
- **Notificaciones**: WhatsApp Mock (Consola) o Twilio.
